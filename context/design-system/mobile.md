# Mobile Design System

## Principles

- consistency first
- reusable components
- predictable spacing
- minimal visual noise

## Typography

Reuse shared typography scale.

## Spacing

Use spacing tokens only:
- xs
- sm
- md
- lg
- xl

Avoid arbitrary spacing values.

## Colors

Use semantic colors:
- primary
- secondary
- success
- warning
- error

Never hardcode colors.

## Components

Prefer reusable:
- buttons
- cards
- list items
- loading states
- error states

## Accessibility

Support:
- dynamic text
- screen readers
- touch target sizing
