# Product Scope Control

## Prime Directive

Do exactly what the user asked.

## Do Not Do Without Explicit Request

- Add features.
- Remove features.
- Change pricing, subscriptions, auth, permissions, user roles, or business logic.
- Change navigation flows.
- Add onboarding flows.
- Add analytics tracking.
- Add external services.
- Change copy in a way that changes meaning.
- Change data schema.

## If Improvement Is Discovered

Do not implement unrelated improvements.

Instead, report them as:

Optional future improvements, not implemented.
