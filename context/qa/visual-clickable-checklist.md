# Visual And Clickable QA Checklist

Use this checklist for browser-based testing.

## Page Load

- page renders without console errors
- loading state works
- final content appears
- no broken images/icons

## Navigation

- links work
- back/forward behavior works
- protected routes behave correctly
- unknown routes behave correctly

## Interaction

- buttons are clickable
- forms accept input
- validation appears
- modals open and close
- dropdowns work
- keyboard navigation works

## Visual

- layout does not overflow
- mobile viewport works
- desktop viewport works
- text is not clipped
- spacing is consistent
- focus states are visible

## Safety

Do not perform destructive production actions during QA unless explicitly requested.
